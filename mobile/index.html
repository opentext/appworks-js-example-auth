AppWorks Auth Demo

You have a few buttons below which perform authentication in different scenarios

"Authenticate" will check the session is still valid, and ifit is, will get the authentication object. If it is not, it will perform a reauthentication, and return the object

"Force Authentication" will perform a reauthentication, and return the authentication object

"Is Online" will return true if the app is connected to the same network as the gateway or false if not.

"OTDS SSO Ticket" will get the OTDS SSO Ticket, if it is available, if not, it will request one using the OAuth token.

Result will appear here.
